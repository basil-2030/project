Course Specification: Human-Computer Interaction

General Information

Course Title: Human-Computer Interaction

Course Code: HCI 2204

Course Learning Outcomes

Code Outcome Description
1.1 Understand the fundamental principles of Human‑Computer Interaction.
1.2 Apply usability principles to design effective user interfaces.
1.3 Evaluate user interfaces based on usability and accessibility standards.

Course Topics

  1. Introduction to HCI Principles
  2. User-Centered Design Process
  3. Interaction Design Models
  4. Usability Testing and Evaluation
